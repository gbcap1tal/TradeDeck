## Packages
recharts | Financial charting library for stock history
framer-motion | Smooth layout transitions and entry animations
date-fns | Date formatting for charts and news timestamps
clsx | Utility for conditional class names
tailwind-merge | Utility for merging tailwind classes

## Notes
- Using Recharts for all financial visualization (AreaChart, BarChart)
- Dark mode is the default and primary aesthetic
- Auth flow uses /api/login and /api/logout provided by Replit Auth
- Stock data APIs are assumed to be available at /api/stocks/*
- WebSocket updates are not implemented in this version (polling used via React Query)
